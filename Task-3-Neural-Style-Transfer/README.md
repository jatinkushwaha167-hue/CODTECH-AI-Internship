# Task 3: Neural Style Transfer

This task demonstrates Neural Style Transfer using deep learning.

## Description
A content image and a style image are combined using a neural network
to generate a stylized output image.

## Files
- neural_style_transfer.py : Python implementation
- content.jpg : Original content image
- style.jpg : Style reference image
- output.png : Generated stylized image

## Output
The final stylized image is saved as output.png.
